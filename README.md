# 🏆 Sistema de Gestión Mundial de Fútbol

Proyecto desarrollado para la materia **Programación Orientada a Objetos** — Taller 2026.

## Descripción

Sistema en Java para centralizar la información deportiva y logística de un torneo de fútbol mundial. Permite gestionar sedes, selecciones, jugadores, partidos y eventos en tiempo real.

## Funcionalidades

- Gestión de sedes y estadios
- Administración de países, selecciones y jugadores
- Organización de grupos, fases y partidos
- Registro de eventos en tiempo real (goles, tarjetas, etc.)
- Informes: tabla de posiciones, ranking de goleadores, informe disciplinario, ficha técnica de partido y estadísticas de sedes
